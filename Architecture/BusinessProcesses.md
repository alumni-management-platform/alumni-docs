# Alumni Management Platform - Business Processes

## Process 1 - Alumni Registration

Actor:
Alumni

Steps:

1. Alumni receives registration link.
2. Alumni logs in.
3. Alumni verifies identity.
4. Alumni updates profile.
5. System stores update request.
6. Admin reviews request.
7. Admin approves or rejects.
8. Alumni receives notification.

---

## Process 2 - Bulk Alumni Upload

Actor:
Data Administrator

Steps:

1. Upload Excel or CSV file.
2. System validates structure.
3. System validates mandatory fields.
4. System identifies duplicates.
5. User reviews validation results.
6. User confirms upload.
7. System imports records.
8. Audit entry created.

---

## Process 3 - Alumni Profile Update

Actor:
Alumni

Steps:

1. Login.
2. Open profile.
3. Update information.
4. Submit change request.
5. Change enters approval queue.
6. Admin reviews.
7. Approve or reject.
8. System updates master record.

---

## Process 4 - Search Alumni

Actor:
Administrator

Steps:

1. Open alumni search.
2. Enter filters.
3. Execute search.
4. View results.
5. Open profile.

---

## Process 5 - Dashboard Analytics

Actor:
Administrator

Steps:

1. Open dashboard.
2. Select report.
3. System aggregates data.
4. Dashboard displays charts.

---

## Process 6 - Audit Review

Actor:
Administrator

Steps:

1. Open audit screen.
2. Search records.
3. Review change history.
4. Export audit report.
